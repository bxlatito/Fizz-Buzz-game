Rules of the Game: FizzBuzz Challenge
A random number between 1 and 100 appears in the display area. For each number you must choose the correct option before the 10 second timer expires.

Button Options:
• Normal
Choose this if the number is not divisible by 3 or 5.
• Fizz
Choose this if the number is divisible by 3 but not by 5.
• Buzz
Choose this if the number is divisible by 5 but not by 3.
• Fizz Buzz
Choose this if the number is divisible by both 3 and 5.

Time Limit:
You have 7 seconds to select the correct option for each number. If you choose the wrong option or fail to answer in time the game ends.
